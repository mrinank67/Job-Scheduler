# **App Name**: ChronoPrint

## Core Features:

- Schedule Definition: Define job schedules (hourly, daily, weekly) through a user-friendly interface.
- Schedule Visualization: Visualize schedules, including start time, interval, and recurrence.
- Job Execution: Trigger "Hello World" console output (or file write) when the scheduled time is reached.
- Recurring Schedules: Implement recurring schedules with options to set specific days of the week or month.
- Job execution log: Include a log to show the job executions.

## Style Guidelines:

- Primary color: Saturated purple (#A64AC9) for a sense of reliability.
- Background color: Light purple (#EEE2DC), a desaturated version of the primary color to maintain visual connection.
- Accent color: Blue (#126E82), used for highlighting interactive elements and CTAs to ensure clarity and engagement.
- Headline font: 'Space Grotesk' sans-serif, ideal for titles and concise information display.
- Body font: 'Inter' sans-serif, ensuring legibility and a modern aesthetic across all devices.
- Minimalist icons representing scheduling, time, and tasks. Use a consistent style.
- A clear layout to display schedules and set up new jobs. Clean and understandable interface to ensure an accessible experience.
